繁體中文排版有別於西文，在字距、行距、標點處理上有其特有的美學邏輯。本文整理了十個容易被忽略的排版細節，從字距設定到標點避頭尾，一次看懂繁中排版的眉角。

## 1. 字距不等於字間距

漢字的字距設定與西文大相徑庭。通常繁體中文正文字距設為 `0` 即可，過多的字距反而顯得鬆散。

> 「方塊字的灰度已經由字形本身決定，再加字距只會破壞節奏。」
> — 柯志杰《字型散步》

## 2. 行距的黃金比例

行距（line-height）的建議值取決於字級與使用場景：

- **長文閱讀**：字級 × 2.0（最舒適）
- **標題排版**：字級 × 1.3 – 1.5
- **短句引文**：字級 × 1.8

## 3. 標點符號的處理

全形標點（。，、！？）在行首或行尾的處理，是繁體中文排版的重要細節：

1. **避頭尾**：「」『』（）不可出現在行首；，。、！？不可出現在行尾
2. **半形化**：數字、英文前後的標點可酌情使用半形
3. **間距控制**：連續標點應視覺調整，避免視覺空洞

## 4. 橫直排混排

橫排文字中若需插入直排元素，需特別注意：

- 使用 `writing-mode: vertical-rl` 控制文字方向
- 數字與英文字母的旋轉設定
- 標點符號在直排時的特殊字符（例如「，」應轉為「、」的視覺位置）

## 5. 字重階層的運用

繁體中文字體的字重選擇有限，通常只有 `Light / Regular / Bold` 三級。善用字重差異建立視覺層級，比單純放大字級更為優雅。

## 6. 中英文混排的字級調整

西文字與漢字在相同 `font-size` 下，視覺大小不同。中英混排時，可透過 CSS 的 `font-size-adjust` 或針對英文字設定獨立字級，讓兩者視覺平衡。

```css
.mixed-text {
  font-family: 'Cormorant Garamond', 'Noto Serif TC', serif;
  font-size: 1rem;
  font-size-adjust: 0.5;
}
```

## 7. 首行縮排的文化習慣

中文段落傳統使用「首行縮排兩字」作為段落標示，而非西式的段間距。現代網頁設計則多改用段落間距，兩種做法各有美感。

## 8. 數字的半形使用

阿拉伯數字一律使用半形，並在前後加入適當空白。全形數字（０１２３）僅在特殊情境（如書法風格）使用。

## 9. 分隔線與留白

繁體中文排版重視「氣韻」，留白是視覺呼吸的關鍵。分隔線不宜過粗，建議 `0.5px – 1px` 的淺色細線即可。

## 10. 字體的文化語境

不同字體承載不同的文化氣質：

- **宋體**：正式、文學、書卷氣
- **黑體**：現代、俐落、商務
- **楷體**：傳統、手寫、親切
- **手寫風格**：溫度、個性、情感

選字體不只是美學決定，更是語境選擇。
