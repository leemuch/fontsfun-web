品牌識別不只是 Logo，而是一套完整的視覺語言系統。從接案到交件，一個完整的品牌專案通常需要 6 到 12 週的時間，本文分享我自己慣用的六階段工作流程。

## 第一階段：品牌策略研究

在動手設計之前，先深入了解客戶的品牌定位、目標受眾、競品分析，這是整個品牌識別系統的基礎。我會準備一份「品牌簡報問卷」（brand brief），請客戶回答以下核心問題：

1. 品牌存在的理由是什麼？
2. 服務的對象是誰？他們有什麼樣的價值觀？
3. 三個能代表品牌氣質的形容詞是什麼？
4. 五年後希望品牌變成什麼樣子？
5. 競品中有哪些是你欣賞或想避免的？

> 「品牌策略不是行銷話術，是品牌與世界對話的姿態。」

## 第二階段：視覺方向提案

收集完策略資訊後，我會準備 **三個不同方向** 的視覺概念，每個方向包含：

- 色彩情緒板（mood board）
- 字體搭配建議（中英文各 1–2 組）
- Logo 草圖方向（手繪 + 數位概念稿）
- 視覺風格參考圖

提案時不直接說「哪個最好」，而是讓客戶從策略角度選擇——哪個方向最能說出他們想說的話。

## 第三階段：Logo 精修與系統雛形

確認方向後，進入 Logo 細部修整階段：

- 字體微調（custom letterform）
- 比例與留白測試
- 黑白、單色、多色版本
- 最小使用尺寸測試
- 不同背景下的可讀性

## 第四階段：設計系統建立

Logo 只是起點，完整的設計系統還包含：

| 系統元素 | 內容 |
|---------|------|
| 色彩 | 主色 / 副色 / 強調色 / 中性色 |
| 字體 | 標題字 / 內文字 / 數字字體 |
| 圖形語言 | 圖示風格 / 插圖風格 / 圖紋 |
| 攝影風格 | 色調 / 構圖 / 光線氛圍 |
| 排版規範 | 字級階層 / 行距 / 對齊邏輯 |

## 第五階段：應用與延伸

設計系統建立後，套用到實際載體：

- 名片、信封、信紙
- 網站視覺、社群貼文模板
- 包裝、招牌、空間視覺
- 員工識別、活動宣傳物

## 第六階段：品牌指南交付

最終交付的不只是檔案，而是一本 **品牌指南手冊**（Brand Guidelines），確保未來其他設計師接手時，能維持品牌的一致性。

```
brand-guideline.pdf
├── 01. Brand Story
├── 02. Logo System
├── 03. Color Palette
├── 04. Typography
├── 05. Imagery
├── 06. Applications
└── 07. Don'ts
```

---

品牌識別的核心，是「一致性」與「靈活性」之間的平衡。好的設計系統應該像一套樂譜——既有結構，又能演奏出多種變奏。
